# API Designer Examples

Per esplorare il Designer nella Milestone 1 puoi aprire questo workspace:

- `docs/examples/api_designer/payments_demo_workspace/workspace.yaml`

Il workspace contiene:

- una API di esempio con piu' path e operation
- una struttura diretta `API -> paths -> operations`, senza nodi artificiali intermedi
- componenti locali
- una shared library
- release definitions
- un change con relativi `ChangeStep`

Aprilo dal Designer con `Open` e seleziona il file `workspace.yaml`.
